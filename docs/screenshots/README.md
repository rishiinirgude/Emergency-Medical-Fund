# Screenshots

Place the following screenshots in this folder:

## Required

### `mobile-view.png`
A screenshot of the app on a mobile screen (375px width).
- Open Chrome DevTools → Toggle device toolbar → iPhone SE or similar
- Capture the home page showing the campaign list
- Referenced in the main README as proof of mobile responsiveness

### `cicd-pipeline.png`
A screenshot of a successful GitHub Actions run.
- Go to your repo → Actions tab → click a passing workflow run
- Screenshot the full job summary showing all green checkmarks
- Referenced in the main README

## Optional (but recommended)

- `home-desktop.png` — Home page on desktop
- `campaign-detail.png` — Campaign detail page
- `admin-panel.png` — Admin hospital management page
- `create-campaign.png` — Create campaign form
